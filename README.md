# Task 3: Zepto Sales & Inventory Dashboard

## 📌 Objective
The objective of this project is to analyze Zepto's sales and inventory data using Power BI and create an interactive dashboard that provides meaningful business insights for better decision-making.

---

## 📂 Dataset
**Dataset:** Zepto Sales & Inventory Dataset

The dataset contains information related to product categories, sales, quantity, discounts, MRP, available quantity, and out-of-stock products.

---

## 🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel

---

## 📊 Dashboard Features

The dashboard includes:

- KPI Cards
  - Total Sales
  - Total Quantity Sold
  - Total Discount
  - Total Out-of-Stock Products

- Sales by Category
- Available Quantity by Category
- Out-of-Stock Products by Category
- Sales vs Quantity Analysis
- Discount vs MRP Analysis
- Interactive Filters

---

## 📈 Key Insights

- Chocolates & Candies recorded the highest sales.
- Cooking Essentials and Munchies experienced the highest stock-outs.
- Categories with higher stock availability generated better sales.
- Higher discounts did not always result in higher sales.
- Low-performing categories may require inventory optimization.

---

## 💡 Business Recommendations

- Improve inventory planning for high-demand categories.
- Reduce stock-outs through better demand forecasting.
- Review discount strategies to maximize profitability.
- Focus inventory investment on high-performing categories.
- Monitor category performance regularly for data-driven decisions.
